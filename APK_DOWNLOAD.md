# APK Download Location

## 📱 Where to Find Your APK

### 1. **Expo Dashboard (Online)**

Build complete hone ke baad:

**URL:**
```
https://expo.dev/accounts/[your-username]/projects/neurodetect-ai/builds
```

**Steps:**
1. Expo dashboard par jao
2. Your projects → `neurodetect-ai` → Builds
3. Latest build click karo
4. **"Download"** button click karo
5. APK file download ho jayegi

---

### 2. **Local Download Location**

APK download hone ke baad:

**Mac:**
```
~/Downloads/neurodetect-ai-*.apk
```

**Windows:**
```
C:\Users\[YourName]\Downloads\neurodetect-ai-*.apk
```

**Linux:**
```
~/Downloads/neurodetect-ai-*.apk
```

---

### 3. **Build Status Check**

Terminal mein build status check kar sakte ho:

```bash
eas build:list
```

Yeh command:
- All builds dikhayega
- Status dikhayega (in-progress, finished, error)
- Download link bhi dikhayega

---

### 4. **Email Notification**

Build complete hone par:
- Expo email bhejega
- Email mein download link hoga
- Direct download kar sakte ho

---

## 📋 Quick Steps After Build:

1. **Wait for build** (15-20 minutes)
2. **Check terminal** - "Build finished" message
3. **Go to Expo dashboard** - https://expo.dev
4. **Navigate to builds** - Your project → Builds
5. **Click Download** - APK file download
6. **Transfer to Android** - USB/Email/Cloud
7. **Install APK** - Allow from Unknown Sources

---

## 🔍 Find Build URL:

Terminal mein yeh command se bhi check kar sakte ho:

```bash
eas build:list
```

Yeh sabhi builds dikhayega with download links.
