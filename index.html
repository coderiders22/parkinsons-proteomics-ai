iOS Installation Guide

iOS Device

Scan to download Expo Go

Download Expo Go
1
Download Expo Go
Install "Expo Go" from the App Store on your iOS device
2
Scan QR Code
Open your iPhone camera and scan the QR code shown on the left
3
Connect to Server
When prompted, connect to the development server (we will grant access)
4
Launch Application
The app will load automatically inside Expo Go and you're ready to use it
Important Note
iOS users require Expo Go to run the application. This ensures seamless updates and compatibility across all iOS versions.

Android Installation Guide

Android Device

Scan to download APK

Direct Download
1
Scan QR Code
Use your Android camera to scan the QR code. The APK will start downloading automatically
2
Enable Unknown Sources
Go to Settings → Security → Enable "Install from Unknown Sources"
3
Install APK
Open the downloaded APK file and tap "Install"
4
Launch & Use
Open NeuroDetect AI normally — no Expo required for Android
Advantage
Android users get the standalone APK — no additional apps required. Install once and use forever with automatic updates.
